**Critical implementation notes:**
- JSON output agents (flashcard, quiz) MUST include "Return ONLY valid JSON, no markdown fences, no explanation" in their instructions. Without this, LLMs frequently wrap output in prose or code fences, breaking JSON parsing.
- All three agents use `get_model()` — this is what makes AGENT-02 work.
- The persona string from PERSONAS is prepended to the task-specific instructions, separated by a double newline.

Write a comprehensive study guide that covers all major concepts.

Create the Agno Workflow subclass that orchestrates the three agents in sequence and yields progress RunResponse objects between each step.

**Key implementation considerations from research:**

1. **RunEvent enum validation:** Before using `RunEvent.running`, inspect the actual enum values available in the installed agno package:
   ```python
   from agno.run.response import RunEvent
   print([e.name for e in RunEvent])
   ```
   Use `workflow_started` for intermediate progress events if `running` does not exist. The frontend reads `content`, not `event`.

2. **Sync/async mismatch:** `Workflow.run()` is a synchronous generator. The SSE endpoint is async. Do NOT call `await asyncio.sleep(0)` inside the generator itself — that belongs in the async wrapper in the SSE endpoint (next plan). The workflow is pure sync here.

3. **JSON parsing with try/except:** Flashcard and quiz agent output is JSON strings. Parse in the workflow and handle parse failures gracefully.

4. **source_title extraction:** Extract from the first heading in the content (look for lines starting with `# `) or fall back to truncating the URL or the first 60 chars of content.

**backend/app/workflows/session_workflow.py:**
```python
import json
import re
from typing import Iterator

from agno.workflow import Workflow
from agno.run.response import RunResponse, RunEvent
from agno.agent import Agent

from app.agents.notes_agent import build_notes_agent
from app.agents.flashcard_agent import build_flashcard_agent
from app.agents.quiz_agent import build_quiz_agent


def _extract_title(content: str, url: str = "") -> str:
    """Extract a title from the first markdown heading or first line of content."""
    for line in content.splitlines():
        line = line.strip()
        if line.startswith("# "):
            return line[2:].strip()
        if line and not line.startswith("#"):
            return line[:80]
    return url[:80] if url else "Untitled"


def _parse_json_safe(raw: str, fallback: list) -> list:
    """Parse JSON from agent output, stripping markdown fences if present."""
    # Strip common markdown fence wrappers
    cleaned = re.sub(r"^```(?:json)?\s*|\s*```$", "", raw.strip(), flags=re.MULTILINE)
    try:
        return json.loads(cleaned)
    except (json.JSONDecodeError, ValueError):
        return fallback


class SessionWorkflow(Workflow):
    description: str = "Generate study materials from extracted content"
    notes_agent: Agent
    flashcard_agent: Agent
    quiz_agent: Agent

    def run(  # type: ignore[override]
        self,
        content: str,
        tutoring_type: str,
        focus_prompt: str = "",
        url: str = "",
    ) -> Iterator[RunResponse]:
        input_text = (
            f"Content:\n{content}\n\nFocus on: {focus_prompt}"
            if focus_prompt
            else f"Content:\n{content}"
        )

        # Step 1: Generate notes
        yield RunResponse(content="Crafting your notes...")
        notes_result = self.notes_agent.run(input_text)
        notes = notes_result.content or ""

        # Step 2: Generate flashcards
        yield RunResponse(content="Making your flashcards...")
        flashcard_result = self.flashcard_agent.run(input_text)
        flashcards_raw = flashcard_result.content or "[]"
        flashcards = _parse_json_safe(flashcards_raw, [])

        # Step 3: Generate quiz
        yield RunResponse(content="Building your quiz...")
        quiz_result = self.quiz_agent.run(input_text)
        quiz_raw = quiz_result.content or "[]"
        quiz = _parse_json_safe(quiz_raw, [])

        # Final: complete event with all data
        yield RunResponse(
            event=RunEvent.workflow_completed,
            content={
                "source_title": _extract_title(content, url),
                "tutoring_type": tutoring_type,
                "notes": notes,
                "flashcards": flashcards,
                "quiz": quiz,
            },
        )


def build_workflow(tutoring_type: str) -> SessionWorkflow:
    """Factory: creates SessionWorkflow with agents configured for the given tutoring type."""
    return SessionWorkflow(
        notes_agent=build_notes_agent(tutoring_type),
        flashcard_agent=build_flashcard_agent(tutoring_type),
        quiz_agent=build_quiz_agent(tutoring_type),
    )
```

**Note on RunResponse constructor:** The `event` parameter may default to `workflow_started` in Agno. Check the installed package. If `RunEvent.workflow_completed` does not exist, use the string `"workflow_completed"` as a fallback — the SSE endpoint checks `response.event` by string comparison in that case.
